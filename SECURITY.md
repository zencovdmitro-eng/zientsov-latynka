# Безпека

Повідомлення про вразливості надсилайте на zencovdmitro@gmail.com.

Не публікуйте закриті ключі, паролі, файли PFX/P12 або коди підтвердження. Офіційні випуски перевіряються SHA-256 і можуть супроводжуватися окремим підписом КЕП Дія. Authenticode-підпис вважається наявним лише після перевірки вбудованого підпису конкретного EXE.
